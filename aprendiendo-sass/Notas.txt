Para poder instalar SASS es necesario primero intalar NODE.JS posteriormente ir a la pagina de sass para ver el codigo que se tiene que escribir en node.js para realizar la insatalacion de sass....(esto es mediante la terminal de windows)

ahora hay que hacer un archivo html y en el escribir la etiqueta link como si fueramos a agregar un archivo css, con el nombre del archivo con .css, en este caso se llama "styles.css"

una vez haciendo lo anterior se debe hacer un fichero de sass, en este caso se llamara "estilos.scss" ese archivo tendra sus respectivas variables y normas de css, una vez listo se debe compilar el fichero de sass, para hacerlo debemos ir a la terminal y escribir la ruta donde se encuentra el archivo de "estilos.scss", en este caso seria de la siguiente manera "cd C:\Users\Danny Marper\Documents\Desarrollo Web\Cursos\CursosUdemy\VictorRobles\MasterEnCSS_VR\aprendiendo-sass", una vez ahi se escribira lo siguiene " sass --watch estilos.scss styles.css", el primer parametro solo indica que es de "sass", el "--watch" indica que cada vez que se haga un cambio en el documento "estilos.scss" este se compilara automaticamente, el tercer parametro es el archivo de entrada y el ultimo es el archivo de salida, osea donde se compilara la informacion de sass, despues dar clic en enter y se estara compilando en tiempo real. y se creara un archivo llamado "styles.css"
